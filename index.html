Hangman

This is a simple hangman game. Try to guess the name of the NBA team. You have 8 tries to guess correctly. Good luck!

How to Play
  1. Press any key to begin
    2. When you press a key, the game starts, and the computer generates a random team name for you to guess from a pre-defined list. You have 8 guesses to guess the word correctly. When you run out guesses, you lose. Game over.
  2. Press a letter on the keyboard. The computer determines if the letter you pressed is in the word.
    • If the letter you guessed is found in the word, you guessed correctly. The letter appears in the appropriate blank(s).
    • If the letter you guessed is not in the word, you guessed incorrectly. The letter is added to the "Letters Already Guessed" section. The number of guesses remaining decreases by 1.
  3. If you correctly guess all the letters in the word before the number of guesses remaining reaches 0, you win.
    4. Else, you lose. Please try again.
